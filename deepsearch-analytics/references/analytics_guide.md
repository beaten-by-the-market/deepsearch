# 분석 함수 & 스크리닝 레퍼런스

## API 기본 정보
- Base URL: `https://api.deepsearch.com/v1/compute?input={쿼리}`
- 인증: `Authorization: Basic {인증키}`

## 1. 스크리닝 조건

### 재무데이터 기반
```
매출 > 1000000000000                   → 매출 1조 이상
매출액 > 1000000000000                 → 동일 (매출/매출액 모두 가능)
영업이익 / 매출 > 0.15                  → 영업이익률 15% 이상
2023 매출 > 2022 매출 * 1.5            → 전년대비 매출 50% 성장
당기순이익 > 0                          → 흑자 기업
```

### 개요데이터 기반
```
설립일 > 2020-01-01                    → 2020년 이후 설립
대표이사 나이 > 70                      → CEO 70세 이상
주거래은행 == "신한은행"                 → 특정 은행 거래
종업원수 > 1000                        → 종업원 1000명 이상
```

### 복합 조건 (and/or)
```
매출 > 1000000000000 and 영업이익 / 매출 > 0.15
"전기차" 관련 기업 and "자율주행" 관련 기업 and 매출액 > 100000000000
```

### 기업 발굴 + 재무 결합
```
("반도체" 관련 기업) and (매출액 영업이익)
(매출 영업이익 당기순이익) in ("건강기능식품" 산업 기업)
```

## 2. 분석 함수

| 함수 | 설명 | 예시 |
|------|------|------|
| `Top(query, n)` | 상위 N개 | `Top(매출액, 10)` |
| `Top(query, n, col)` | 특정 열 기준 상위 N | `Top(매출 > 1조 and 영업이익, 10)` |
| `Bottom(query, n)` | 하위 N개 | `Bottom(시가총액, 10)` |
| `Sort(query, ascending=False)` | 정렬 | `Sort(매출액 > 1조, ascending=False)` |
| `Rank(query)` | 순위 | `Rank(매출액)` |
| `Max(query)` | 최대값 | `Max(삼성전자 매출액 2015-2024)` |
| `Min(query)` | 최소값 | `Min(LG전자 영업이익 2015-2024)` |
| `Mean(query)` | 평균 | `Mean(삼성전자 종가 2024-01-01-2024-12-31)` |
| `Sum(query)` | 합계 | `Sum(Select("반도체" 관련 기업 and 매출액, 1))` |
| `Select(query, idx)` | 컬럼 선택 | `Select(query, [1,2])` 또는 `Select(query, ["매출","영업이익"])` |

### 4칙 연산
```
영업이익 / 매출 > 0.2                  → 영업이익률 20% 이상
2023 매출 > 2022 매출 * 2             → 전년대비 2배 성장
삼성전자 매출액 + LG전자 매출액        → 합산
Max(삼성전자 매출액 2010-2024) / Min(삼성전자 매출액 2010-2024) → 최대/최소 비율
```

## 3. 2단계 분석 패턴

### 패턴 A: 스크리닝 → 상세 조회
1단계: `매출 > 1조 and 영업이익/매출 > 0.15` → 기업 리스트 추출
2단계: 재무 상세 → `GetFinancialStatements(기업명, ...)`, 주가 → `기업명 종가 기간` 자연어 쿼리

### 패턴 B: 뉴스+주가 결합 분석
1단계: `DocumentSearch(...)` → 뉴스 발생 날짜/기업 확인
2단계: `삼성전자 종가 2024-06-01-2024-06-30` → 해당 기간 주가 조회 (주가는 자연어 OK)

### 패턴 C: 포트폴리오 수익률 시뮬레이션
1단계: 스크리닝으로 기업 선정
2단계: `삼성전자 종가 2024-01-02-2024-01-02` → 시작/종료 시점 종가 조회 (주가는 자연어 OK)
3단계: 시작가/종가 비교로 수익률 계산

## 주의사항
- Top() 함수에 복잡한 "관련 기업" 쿼리를 넣으면 실패할 수 있음 → 2단계로 분리
- 재무데이터(연/분기)와 시장데이터(일별)는 별도 쿼리 권장
- 스크리닝 결과가 너무 많으면(10000+) 조건을 더 추가하여 좁히기
