# 가치투자 방법론 & DeepSearch 적용 가이드

## 1. Benjamin Graham 방법론

### 핵심 기준
| 기준 | 조건 | DeepSearch 쿼리 |
|------|------|-----------------|
| 적정 규모 | 매출 1조 이상 | `매출 > 1000000000000` |
| 수익 안정성 | 영업이익률 10%+ | `영업이익 / 매출 > 0.1` |
| 흑자 기업 | 당기순이익 양(+) | `당기순이익 > 0` |
| 저PER | PER < 15 | `{기업명} PER` (개별 조회) |
| 저PBR | PBR < 1.5 | `{기업명} PBR` (개별 조회) |
| Graham Number | PER × PBR < 22.5 | 계산으로 확인 |

### 실행 방법 (2단계)

**1단계 - 우량 기업 스크리닝:**
```
매출 > 1000000000000 and 영업이익 / 매출 > 0.1 and 당기순이익 > 0
```
→ 재무적으로 안정된 대형 기업 리스트 추출

**2단계 - 개별 기업 밸류에이션 확인:**
```
SK하이닉스 PER
SK하이닉스 PBR
삼성전자 PER
삼성전자 PBR
```
→ PER < 15, PBR < 1.5, PER × PBR < 22.5 조건 충족 기업 선별

### 주의사항
- PER/PBR 대량 스크리닝(`PER < 15`)은 API에서 빈 결과를 반환할 수 있음
- 반드시 1단계 재무 스크리닝 후 → 2단계 개별 PER/PBR 조회 방식 사용

## 2. Warren Buffett 방법론

### 핵심 기준
| 기준 | 조건 | DeepSearch 쿼리 |
|------|------|-----------------|
| 경제적 해자 | 높은 영업이익률 | `영업이익 / 매출 > 0.15` |
| 대형 우량주 | 매출 1조 이상 | `매출 > 1000000000000` |
| 이익 성장 | 전년대비 성장 | `2023 매출 > 2022 매출 * 1.1` |
| 꾸준한 수익 | 다년간 흑자 | 연도별 당기순이익 확인 |

### 실행 방법 (2단계)

**1단계 - 높은 마진 대형주 스크리닝:**
```
매출 > 1000000000000 and 영업이익 / 매출 > 0.15
```
→ 약 60개 기업 추출

**2단계 - 이익 지속성 확인 (GetFinancialStatements):**
```
GetFinancialStatements(삼성전자, report_type="IFRS", consolidated=True, report_ids="Income", date_from=2020-01-01, date_to=2024-12-31)
GetFinancialStatements(SK하이닉스, report_type="IFRS", consolidated=True, report_ids="Income", date_from=2020-01-01, date_to=2024-12-31)
```
→ 507개 항목에서 영업이익 필터링하여 5년간 꾸준히 흑자인 기업 선별

## 3. 포트폴리오 수익률 시뮬레이션 (3단계)

### 목적
특정 투자 전략으로 기업을 선정했을 때 실제 수익률이 얼마인지 백테스팅

### 실행 방법

**1단계 - 전략 기반 기업 선정:**
```
매출 > 1000000000000 and 영업이익 / 매출 > 0.15
```

**2단계 - 기간별 주가 조회:**
```
SK하이닉스 종가 2024-01-02-2024-01-02
SK하이닉스 종가 2024-12-30-2024-12-30
삼양식품 종가 2024-01-02-2024-01-02
삼양식품 종가 2024-12-30-2024-12-30
```

**3단계 - 수익률 계산:**
```
수익률(%) = (종가_연말 - 종가_연초) / 종가_연초 × 100
포트폴리오 수익률 = 개별 수익률의 평균 (균등배분 가정)
```

### 결과 포맷
```
## 포트폴리오 백테스팅 결과

전략: Buffett 스타일 (영업이익률 15% 이상 대형주)
기간: 2024-01-02 ~ 2024-12-30

| 기업명 | 종목코드 | 연초 종가 | 연말 종가 | 수익률 |
|--------|---------|----------|----------|--------|
| SK하이닉스 | 000660 | 132,000 | 177,000 | +34.1% |
| 삼양식품 | 003230 | 125,000 | 580,000 | +364.0% |
| ... | ... | ... | ... | ... |

균등배분 포트폴리오 평균 수익률: +XX.X%
KOSPI 동기간 수익률: +X.X% (벤치마크)
```

## 4. 성장주 스크리닝

### 매출 성장률 기반
```
2023 매출 > 2022 매출 * 1.5
```
→ 전년대비 매출 50% 이상 성장 기업 (비상장 포함 약 4,620개)

### 조건 추가로 범위 축소
```
2023 매출 > 2022 매출 * 1.5 and 매출 > 100000000000
```
→ 매출 1,000억 이상이면서 50% 이상 성장 기업

## 5. 배당투자 전략

### 배당 정보 조회
```
GetCompanyDividends("삼성전자",date_from=20200101,date_to=20241231)
```

### 배당수익률 계산 (2단계)
**1단계 - DPS 조회:**
```
삼성전자 DPS
```

**2단계 - 현재가 대비 수익률 계산:**
```
배당수익률(%) = DPS / 현재 종가 × 100
```

## 6. 2단계 분석 패턴 (중요)

DeepSearch에서 복합 분석이 필요한 경우 반드시 2단계로 분리해야 합니다.

### 이유
- 재무데이터(연/분기 업데이트)와 시장데이터(일별 업데이트)는 업데이트 주기가 다름
- 문서 검색과 수치 데이터 조회는 별도 API 영역
- 복잡한 중첩 쿼리는 실패할 수 있음

### 패턴
```
1단계: 조건 기반 기업 리스트 추출 (스크리닝 쿼리)
2단계: 추출된 기업별 상세 데이터 조회 (API 함수 — GetFinancialStatements, SearchTargetPrices 등)
      주가/PER/PBR 등 시장 데이터만 자연어 쿼리 사용
3단계: 결과 종합 및 분석 (계산/비교)
```
