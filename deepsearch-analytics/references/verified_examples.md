## 사례 1: 고수익 대형 기업 스크리닝 (Buffett 스타일)

## 사례 5: 뉴스 + 주가 결합 분석 (2단계)

**자연어:** "삼성전자에서 2024년 6월에 반도체 관련 뉴스가 나왔을 때 주가가 어떻게 변했는지 알려줘"

**1단계 - 뉴스 검색:**
```
DocumentSearch(["news"],["economy"],"securities.name:삼성전자 and 반도체",count=5,page=1,date_from=20240601,date_to=20240630)
```
→ 1,224건의 뉴스 확인, 주요 이벤트 날짜 파악

**2단계 - 주가 조회:**
```
삼성전자 종가 2024-06-01-2024-06-30
```
→ 20거래일 일별 종가 데이터로 뉴스 발생 전후 주가 변동 확인

## 사례 6: Benjamin Graham 가치투자 스크리닝 (2단계)

**자연어:** "벤저민 그레이엄 기준으로 저평가된 우량 기업을 찾아줘"

**Graham 기준:** 매출 안정 + 영업이익 양(+) + 당기순이익 양(+)

**1단계 - 우량 기업 필터:**
```
매출 > 1000000000000 and 영업이익 / 매출 > 0.1 and 당기순이익 > 0
```

**2단계 - 개별 기업 PER/PBR 확인:**
```
SK하이닉스 PER
SK하이닉스 PBR
```
→ PER < 15, PBR < 1.5, PER × PBR < 22.5 조건 충족 기업 선별

## 사례 7: Warren Buffett 스타일 포트폴리오 수익률 시뮬레이션 (3단계)

**자연어:** "영업이익률 15% 이상인 대형주로 2024년 포트폴리오를 구성했다면 수익률이 얼마였을까?"

**1단계 - 기업 선정:**
```
매출 > 1000000000000 and 영업이익 / 매출 > 0.15
```
→ 60개 기업 추출 (KRX 코드 확인)

**2단계 - 주요 기업 주가 조회 (연초~연말):**
```
SK하이닉스 종가 2024-01-02-2024-01-02
SK하이닉스 종가 2024-12-30-2024-12-30
삼양식품 종가 2024-01-02-2024-01-02
삼양식품 종가 2024-12-30-2024-12-30
```

**3단계 - 수익률 계산:**
(종가_연말 - 종가_연초) / 종가_연초 × 100 = 수익률(%)
→ 개별 종목 수익률 및 평균 포트폴리오 수익률 산출

## 사례 14: 뉴스+주가 결합 원인분석 (KRX 업무용, 2단계)

**업무:** 주가급변 종목에 대한 뉴스 기반 원인분석

**1단계 - 뉴스 검색:** (deepsearch-docs 스킬)
```
DocumentSearch(["news"],["economy"],"securities.name:에코프로비엠",count=100,page=1,date_from=20250201,date_to=20250228)
```
→ 69건 뉴스에서 주요 이벤트(자사주 소각, 주가 급변) 파악

**2단계 - 주가+거래량 조회:**
```
에코프로비엠 종가 거래량 2025-02-24-2025-02-28
```
→ 5거래일 일별 종가+거래량으로 뉴스 발생일 전후 이상 변동 확인

**분석 포인트:**
- 뉴스 발생 전 거래량 급증 → 내부자거래 의심
- 부정 뉴스에도 주가 상승 → 추가 조사 대상
